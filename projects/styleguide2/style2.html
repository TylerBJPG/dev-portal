Bin Bois

Official Style Guide

The overall styling of Bin Bois is fairly simplistic however strict adherance to these style guidelines is what achieves this simplistic branding. An example of this is the standard black text logo should only be used on white or light gray backgrounds.

Dark Fuschia

Dark Fuschia is used to accent buttons & navigation. It can also be used to highlight important pieces of text online. It is currently not used in a print capacity.

Soft Mauve

Soft Mauve is used online for buttons and other navigational items; it is currently not used in a print capacity.

True White

True white is used in both an online and print capacity to contrast true black.

True Black

True black is used in both an online and print capacity to contrast true black.
